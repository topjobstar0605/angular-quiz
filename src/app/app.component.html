Exercise-1: Answer Select

Напишите приложение, где работали бы два компонента:

Упражнение select. На вход компонент принимает массив ответов (строка + правильность ответа).

Пример: [(value: "apple", correct: false), (value: "pear", correct: true), (value: "pineapple", correct: false)]

Упражнение выглядит как обычный html select, предлагающий выбрать ответ из списка.

При этом, компонент на страницу может добавляться как через ng-repeat и передачу данных из контроллера, так и вручную.

Упражнения могут располагаться в разных частях страницы и находиться внутри изолированных scope.

Виджет Counter для отображения количества очков. Отображает всего 2 числа: общее количество ответов; количество правильных ответов.

Exercise-2: Translation Pair

Напишите компонент, который реализует виджет для изучения слов.

На вход компонент принимает массив слов и переводов.

Пример: [(word: "apple", translation: "яблоко"), (word: "nuts", translation: "орех")]

Компонент должен выводить на экран в один момент времени только одну пару слов (например, "apple — яблоко"), переключение к следующей паре происходит при нажатии на пробел.
